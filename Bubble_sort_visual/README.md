# Bubble Sort Visualization

This repository contains a small Python script to visualize the bubble sort algorithm step-by-step using matplotlib.

Files
- `bubble_sort_visual.py`: main script that runs the visualization.

Dependencies
- Python 3.8+
- matplotlib

Install
```bash
pip install matplotlib
```

Run
```bash
python bubble_sort_visual.py --size 30 --interval 150
```

Options
- `--size`: number of elements (default 20)
- `--interval`: animation frame interval in ms (default 200)
- `--seed`: random seed
- `--list`: provide a list of integers to sort, e.g. `--list 5 3 8 1`

Enjoy the visualization!
